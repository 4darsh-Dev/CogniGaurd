What is CogniGuard?

CogniGuard is a powerful web extension designed to empower users by identifying and combatting dark patterns on various websites, particularly focusing on E-commerce platforms. Ensuring users a transparent and ethical online experience.

Download for Chrome

Features

Transparency Score

This web extension is powered by Fine-Tuned Deep Learning model BERT for classifiying dark patterns.

Dark Pattern Detection

Dark Pattern it detects: Urgency, Scarcity, Misdirection, Social-Proof, Obstruction, Sneaking, Forced Action + Not Dark Pattern.
